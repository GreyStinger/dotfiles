# Jayden's DotFiles

Configuration for my commonly used programs

## Requirements

At minimum 4 things are required to take advantage of these configs everything
else is very personal

 - Nix 2.18.2 or Higher
 - Git
 - tmux V3.4 or Higher
 - NeoVim V0.10.0 or Higher
